# Zenra Bot for Slack

An example for "[new Slack platform](https://api.slack.com/future)" with Deno.

## Usage

Develop:

```shell
slack run
```

Deploy

```shell
slack deploy
```

Add a trigger:

```shell
slack trigger create --trigger-def ./src/trigger.ts
```
